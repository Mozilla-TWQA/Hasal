Only support Windows 7 and Windows 10 currently. Please download "OBS-Studio-18.0.1-Full-Installer.exe" from https://github.com/jp9000/obs-studio/releases/tag/18.0.1 link, and then install into "C:\Program Files (x86)" folder.
